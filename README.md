# Dshbloks

A minimalistic application with with accessible SolidUI components, powered by [`solid-start`](https://start.solidjs.com);

## Tech Stack

- **Framework:** [SolidStart](https://start.solidjs.com/)
- **UI Components:** [Kobalte](https://kobalte.dev/) - Accessible UI primitives
- **Styling:** [Tailwind CSS v4](https://tailwindcss.com/)
- **Fonts:** [Geist Variable](https://vercel.com/font) via Fontsource
- **Dark Mode:** Built-in with ColorModeProvider

## Getting Started

### Prerequisites

- Node.js >= 22

### Installation

```bash
npm install
```

### Development

```bash
npm run dev
```

Open [http://localhost:3000](http://localhost:3000) in your browser.

### Build

```bash
npm run build
npm start
```

## Features

- ✅ TypeScript support
- ✅ Dark mode with SSR
- ✅ Accessible UI components
- ✅ Custom Tailwind theme
- ✅ Modern font stack (Geist)
